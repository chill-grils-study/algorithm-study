## ✅ How To
📅 2025.03.25 ~

1. 매주 3문제
2. 매주 화요일 ``21:00`` 코드 리뷰
3. 각 주차별 Directory Name: 'Week{num}'
   - num : 2자리 숫자로
   - ex) Week01
5. File Name: ``문제번호_이름``
   - 백준 : 2573_jh.java
6. Commit Message: ``[Solve] ✍ 문제번호.문제명``
   - [Solve] ✍ 2573.빙산
